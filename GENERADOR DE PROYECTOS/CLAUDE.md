## COMPONENTES MGA — ESTRUCTURA ESPERADA

### 1. Identificación
- Descripción del problema central
- Árbol de problemas (causas directas, indirectas / efectos directos, indirectos)
- Árbol de objetivos (medios y fines)
- Análisis de involucrados (beneficiarios, afectados, ejecutores, financiadores)
- Población objetivo y área de influencia

### 2. Preparación y Evaluación de Alternativas
- Descripción de alternativas de solución
- Criterios de selección
- Alternativa óptima seleccionada y justificación

### 3. Programación
- Cadena de valor: Insumos → Actividades → Productos → Resultados → Impactos
- Indicadores (nombre, fórmula, línea base, meta, fuente, periodicidad)
- Plan de adquisiciones
- Cronograma de actividades (fechas de inicio y fin)
- Presupuesto por actividad y rubro (Clasificador Presupuestal)
- Fuentes de financiación

### 4. Evaluación
- Evaluación socioeconómica (VPN social, TIR social, Costo-Beneficio o Costo-Eficiencia)
- Evaluación financiera si aplica (VPN, TIR, B/C)
- Evaluación ambiental (impactos y medidas de mitigación)
- Evaluación de riesgo (amenazas, vulnerabilidades, probabilidad, impacto, medidas)

## INDICADORES — FORMATO ESTÁNDAR MGA

Para cada indicador incluir:
| Campo | Descripción |
|---|---|
| Nombre | Descripción clara del indicador |
| Fórmula | Expresión matemática de cálculo |
| Unidad de medida | Número, porcentaje, ratio, etc. |
| Línea base | Valor actual antes del proyecto |
| Meta | Valor esperado al finalizar |
| Fuente de verificación | Documento o sistema donde se mide |
| Periodicidad | Frecuencia de medición |
| Responsable | Entidad o cargo que reporta |

## PRESUPUESTO — CLASIFICADOR PRESUPUESTAL
Usar rubros vigentes del Ministerio de Hacienda:
- **1. Gastos de Personal** (si aplica)
- **2. Gastos Generales** (servicios, materiales, viáticos)
- **3. Transferencias** (si aplica)
- **4. Inversión** (infraestructura, equipos, consultorías)

## TAREAS FRECUENTES

### Redacción de Componentes
- Redactar en lenguaje MGA: verbos en infinitivo para actividades, indicadores con todos sus campos
- Adaptar terminología a los requerimientos del sistema BPIN
- Coherencia entre árbol de problemas → objetivos → cadena de valor → indicadores

### Revisión de Documentos
- Identificar inconsistencias entre componentes
- Verificar coherencia de la cadena de valor
- Revisar que los indicadores tengan línea base y meta medibles
- Detectar riesgos no contemplados

### Evaluación Económica
- Calcular VPN, TIR, B/C con los datos proporcionados
- Definir tasa social de descuento según lineamientos DNP (12% anual referencia)
- Estructurar flujo de costos y beneficios

### Gestión de Riesgos PMI
- Identificar riesgos por categoría (técnico, externo, organizacional, de gestión)
- Valorar probabilidad (1-5) e impacto (1-5)
- Definir respuesta: evitar, transferir, mitigar, aceptar
- Calcular exposición al riesgo y reserva de contingencia

### Cronograma
- Estructurar actividades con predecesoras y duraciones
- Identificar ruta crítica
- Formato compatible con MS Project o Excel
